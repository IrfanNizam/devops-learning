# DevOps Learning Repository

This repository is a **hands-on DevOps learning workspace**.  
It contains practical exercises and small projects based on the **DevOps Roadmap by roadmap.sh**.

The goal is not to build a single large product, but to **systematically practice core DevOps concepts** in isolation, the same way they are encountered in real-world environments.

---

## Purpose

- Practice DevOps fundamentals through focused, self-contained exercises
- Reinforce understanding of systems, infrastructure, and automation
- Build muscle memory for tools commonly used in production
- Track personal progress across the DevOps roadmap

## Topics Covered

Depending on progress, this repository may include exercises related to:

- Linux fundamentals
  - User and group management
  - Permissions and processes
  - System monitoring and troubleshooting
- Networking basics
  - Ports, DNS, HTTP, TLS
- Version control
  - Git workflows and best practices
- CI/CD
  - Pipelines, automation, reusable workflows
- Containers
  - Docker basics, image creation, container lifecycle
- Configuration management
  - Ansible fundamentals
- Infrastructure as Code
  - Terraform basics
- Cloud and reliability concepts
  - Observability, logging, metrics

Not all topics are implemented at once; this repository evolves over time.
